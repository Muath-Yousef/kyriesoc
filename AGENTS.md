# SOC Root Premium UI/UX Specialization

When modifying the frontend, adhere to these "Cyber-Enterprise" design principles:

1. **Cyber-Gold Aesthetic**: 
   - Background: Deep Space Black (`#05050A`)
   - Primary: Neon Cyan (`#00f0ff`)
   - Accent: Amber/Gold (`#fbbf24`) for high-value CTA/alerts.
   - Contrast: Use `white/10` borders for a "blueprint" technical feel.

2. **Glassmorphism**: Always use `backdrop-blur-md` with `bg-white/5` or `bg-black/50` for overlays and cards.

3. **Micro-Animations**: Use `framer-motion` for every section entry. Use subtle scale/glow transitions on group hover.

4. **Typography**: Strictly use `font-sans` (Inter/Outfit) for body and `font-mono` for technical labels, versioning, and status indicators.

5. **Visual Hierarchy**: 
   - Section spacing: `py-24` to `py-32`.
   - Card padding: `p-8`.
   - Headlines: Large, tight tracking, with subtle gradients.
